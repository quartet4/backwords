The Code for Backwords 

            
              // Get string from the page
// Controller
function getValue() {
  document.getElementById("alert").classList.add("invisible");

  let userString = document.getElementById("userString").value;

  let revString = reverseString(userString);

  displayString(revString);
}

// Reverse the string
// Logic
function reverseString(userString) {
  let revString = [];

  // reverse string using a for loop
  for (let i = userString.length - 1; i >= 0; i--) {
    revString += userString[i];
  }

  return revString;
}

// Display the reversed string to the user
// View
function displayString(revString) {
  // write to the page
  document.getElementById("msg").innerHTML = `<strong>${revString}</strong>`;

  // show the alert box
  document.getElementById("alert").classList.remove("invisible");
}

The code is structured in three functions: controller, logic, and view.

getValue()

This is called when the user clicks the "Reverse Me!" button on the form. It hides the alert box, gets the string in the text input, then calls the logic function to get the reversed string, and finally calls the view function to display it in the alert box.

reverseString(userString)

This function reverses the userString parameter via a reverse traversal of the string array using a for loop, then returns the reversed string.

displayString(revString)

This function puts the revString in the alert box then removes the invisible class from the alert box so it becomes visible to the user.
